# 小红书笔记发布

## 前提条件
- Chrome 浏览器已安装
- OpenClaw Chrome 扩展已连接
- 用户已登录小红书创作服务平台

## 操作流程

### 1. 打开发布页面
使用 browser 工具打开：`https://creator.xiaohongshu.com/publish/publish?source=official`

### 2. 选择发布方式
- 点击「写长文」（纯文字）
- 或点击「上传图文」（图片+文字）

### 3. 填写内容
- 标题：输入标题（最多64字）
- 正文：点击下方文本框输入内容

### 4. 一键排版（可选）
- 点击「一键排版」按钮
- 选择模板（如「简约基础」）

### 5. 下一步
- 点击「下一步」

### 6. 封面设置（可选）
- 选择模板封面

### 7. 发布
- 点击「发布」按钮

## 注意事项
- 需要用户已登录小红书账号
- Chrome 扩展需要连接才能操作
- 发布成功后记录到记忆
