# Industry News Tracker - 行业新闻动态追踪

## 技能概述

本技能用于追踪指定关键字相关的：
- 行业新闻动态
- 技术发展趋势
- 展会/论坛/发布会信息
- 解决方案和应用案例

## 使用方法

### 基本搜索

用户提供关键字列表后，执行以下步骤：

1. **多数据源抓取** - 从多个网站获取相关内容
2. **内容提取** - 使用 web_fetch 抓取详细页面
3. **摘要整理** - 提取关键信息和原文链接
4. **格式输出** - 按类别整理，带链接

### 关键字列表

用户可能提供的关键字包括但不限于：
- 工业互联网、智能制造、工业软件
- 设备全生命周期、设备管理、设备全寿命周期
- 设备完整性管理、EAM、设备管理系统
- 资产管理系统、资产全生命周期管理
- 设备管理软件、设备预测性维护
- 设备故障与健康管理、预测性维护
- 设备健康管理系统、设备状态监测
- AI+设备管理、声振温监测、声振温
- 声振温系统、AI赋能制造业
- 企业数字化转型、制造业数智化

```markdown
| 活动名称 | 时间 | 地点 | 主题 |
|---------|------|------|------|
| xxx | xxx | xxx | xxx |

### 三、趋势总结

| 领域 | 热点方向 | 数据源 |
|------|---------|--------|
| xxx | xxx | xxx |
```

## 输出要求

1. **必须带链接** - 每条新闻必须附带原文链接，方便用户查看详情
2. **多数据源** - 不要局限单一网站，至少从5个以上不同来源收集
3. **广东省展会** - 重点关注深圳、广州、佛山、东莞等地的工业展会
4. **分类清晰** - 按主题类别整理内容
5. **保持更新** - 关注最新动态，时间优先

## 报告结构要求

### 行业新闻部分
- 至少包含5个数据源的内容
- 国内网站优先
- 每条新闻不超过3句话摘要

### 展会部分
- 重点收集广东省内的工业展会信息
- 包含：展会名称、时间、地点、主题、官网链接

### 趋势总结
- 按领域分类
- 标注数据来源

## 配置文件

如需持久化关键字配置，可在用户目录创建 `industry-keywords.json`：

```json
{
  "keywords": [
    "工业互联网",
    "智能制造",
    "设备预测性维护"
  ],
  "dataSources": [
    "gongkong.com",
    "iteschina.com",
    "e-works.net.cn"
  ]
}
```

## 注意事项

- 搜索API不可用时，使用 web_fetch 直接抓取网站内容
- 优先选择有实质内容的页面进行深度抓取
- 展会信息重点关注时间、地点、主题、官网链接
- 保持报告简洁，每条新闻不超过3句话摘要
- 广东省展会信息必须包含深圳、广州、佛山、东莞等地
