# 公司背景调研简报

根据用户提供的公司名称，自动搜索并生成公司背景简报，用于销售前期调研。

---

## 使用方式

### 快速开始

```
用户：给我生成XXX公司的简报
助手：（调用此skill，搜索公司信息并生成简报）
```

### 触发方式

用户说以下任意一句时调用此 skill：
- 给我 XXX 公司的简报
- 生成 XXX 公司背景调研
- XXX 公司情况介绍一下
- 给我一份 XXX 的客户简报
- 做一份 XXX 公司的背调
- **做 XXX 公司的背调简报**
- XXX 客户背景调研

### 调用流程

1. 接收公司名称
2. 使用 Tavily API 搜索公司信息
3. 提取关键字段（见下方字段清单）
4. 生成 Markdown 格式简报
5. 保存到 docs/ 目录

---

## 功能特性

### 1. 多源数据采集

| 方式 | 优先级 | 说明 |
|------|--------|------|
| **Tavily AI** | ⭐⭐⭐ | AI 优化的搜索和摘要（推荐） |
| **Web Fetch** | ⭐⭐ | 直接获取网页内容 |
| **百度搜索** | ⭐ | 备用搜索源 |

### 2. AI 增强信息提取

- **OpenAI GPT** 智能解析搜索结果
- 自动提取：公司概况、主营业务、产品服务
- 生成：销售建议、切入话题、客户痛点

### 3. 智能输出

- Markdown 格式简报
- 飞书消息摘要
- 销售沟通建议

---

用户输入

## ⚠️ 重要：输出要求

### 1. 本地保存
- Markdown 文件保存至: `D:\ai-openclaw\docs\{公司名简写}-客户简报-{YYYYMMDD}.md`

### 2. 用户通知（必须）
- **禁止**：仅保存在本地不通知用户
- **必须**：生成简报后，在当前会话中直接展示给用户
- 如果飞书发送失败（需要 openId），则在当前会话窗口展示完整简报
- 简报应包含核心要点：成立时间、总部、业务、销量/营收、战略方向、销售建议

### 3. 用户展示格式

生成简报后，必须在当前会话中向用户展示，格式如下：

```markdown

- 切入话题: XXX

📁 完整简报: {文件路径}
```

**注意**：如果飞书发送失败，必须在当前会话窗口展示上述格式的简报摘要。

## 简报内容结构

### 必含章节

1. **公司概况** - 基本信息（成立时间、注册资本、性质、地址、联系方式、官网）
2. **主营业务** - 核心产品和解决方案
3. **行业地位** - 资质认证、市场排名、技术优势
4. **近期动态** - 新闻、融资、重大事件
5. **客户群体** - 行业分布、典型客户
6. **联系方式** - 电话、邮箱、地址
7. **销售建议** - 切入话题、客户痛点、竞争优势

### 公司概况字段清单

| 字段 | 说明 | 示例 |
|------|------|------|
| 公司名称 | 全称 | 广东中设智控科技股份有限公司 |
| 简称 | 常用简称 | 中设智控 |
| 股票代码 | 如有上市 | 002906 |
| 英文名 | 英文名称 | Foryou Corporation |
| 成立时间 | 精确到年月日 | 1999年9月7日 |
| 法定代表人 | 法人代表 | 郭智聪 |
| 注册资本 | 含币种 | 12108.228万人民币 |
| 企业性质 | 国企/民企/合资/上市 | 国企控股混合所有制 |
| 归属 | 母公司/集团 | 科学城（广州）投资集团 |
| 注册地址 | 工商注册地址 | 广州市黄埔区科珠路233号 |
| 经营地址 | 实际办公地址 | 广州市黄埔区开创大道2399号至泰广场A6栋9层 |
| 联系电话 | 电话/400/传真 | 400-100-4168 |
| 官网 | 公司官网 | www.kingtangdata.com |

### 输出文件

- **Markdown**: `docs/{公司名简写}-客户简报-{YYYYMMDD}.md`
- **飞书消息**: 简报摘要

## 最佳实践

### 数据来源优先级

1. **Tavily AI** - AI 优化摘要，质量最高
2. **企业官网** - 第一手信息
3. **天眼查/企查查** - 官方工商数据
4. **百度搜索** - 通用信息

### 信息核验

- 注册资本以官方数据为准
- 新闻时效性检查（优先近1年）
- 多源交叉验证关键信息

### 搜索技巧

1. **全称搜索**：用户给全称时，直接用全称搜索，不修改
2. **简称匹配**：用户给简称时，根据搜索结果匹配正确公司
3. **多词组合**：用"公司名称 + 主营"组合搜索提高准确度
4. **英文名**：可同时搜索英文名作为补充

### 格式要求

- 公司名称使用全称
- 金额统一为万元/亿元
- 日期格式：YYYY年MM月DD日
